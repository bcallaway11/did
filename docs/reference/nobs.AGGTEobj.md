# Number of unique cross-sectional units in an AGGTEobj object

Number of unique cross-sectional units in an AGGTEobj object

## Usage

``` r
# S3 method for class 'AGGTEobj'
nobs(object, ...)
```

## Arguments

- object:

  a model of class AGGTEobj produced by the
  [`aggte()`](https://bcallaway11.github.io/did/reference/aggte.md)
  function

- ...:

  additional arguments (ignored)

## Value

Integer. The number of unique cross-sectional units in the data.
