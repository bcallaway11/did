# Number of unique cross-sectional units in an MP object

Number of unique cross-sectional units in an MP object

## Usage

``` r
# S3 method for class 'MP'
nobs(object, ...)
```

## Arguments

- object:

  a model of class MP produced by the
  [`att_gt()`](https://bcallaway11.github.io/did/reference/att_gt.md)
  function

- ...:

  additional arguments (ignored)

## Value

Integer. The number of unique cross-sectional units in the data.
