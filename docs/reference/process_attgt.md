# Process Results from [`compute.att_gt()`](https://bcallaway11.github.io/did/reference/compute.att_gt.md)

Process Results from
[`compute.att_gt()`](https://bcallaway11.github.io/did/reference/compute.att_gt.md)

## Usage

``` r
process_attgt(attgt.list)
```

## Arguments

- attgt.list:

  list of results from
  [`compute.att_gt()`](https://bcallaway11.github.io/did/reference/compute.att_gt.md)

## Value

list with elements:

- group:

  which group a set of results belongs to

- tt:

  which time period a set of results belongs to

- att:

  the group time average treatment effect
